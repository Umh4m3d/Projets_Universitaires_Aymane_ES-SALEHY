# Autres TP Web

Ce dossier regroupe plusieurs travaux pratiques réalisés dans le cadre du module
**Développement Web et Sécurité Web**.  
Ces travaux ont pour objectif de renforcer la compréhension des bases du web
(HTML, CSS) ainsi que des vulnérabilités courantes des applications web,
notamment les attaques de type **Cross‑Site Scripting (XSS)**.

---

## Objectifs pédagogiques

- Comprendre la structure d’une page web en HTML
- Appliquer des styles de base avec CSS
- Concevoir des pages web simples et fonctionnelles
- Découvrir les failles de sécurité web
- Exploiter et analyser les vulnérabilités XSS dans un environnement contrôlé
- Sensibilisation aux bonnes pratiques de sécurité web

---

## Contenu du dossier

### 1️⃣ TP : Création d’une carte d’étudiant (HTML & CSS)

**Description :**  
Ce TP consiste à concevoir une carte d’étudiant simple en utilisant **HTML**
pour la structure et **CSS** pour la mise en forme.

**Objectifs spécifiques :**
- Utilisation des balises HTML de base
- Mise en page avec CSS
- Amélioration de l’aspect visuel d’une page web

**Technologies utilisées :**
- HTML
- CSS

---

### 2️⃣ TP : Exploitation des failles XSS avec DVWA

**Description :**  
Ce TP vise à comprendre et exploiter les différentes formes de vulnérabilités
**XSS (Cross‑Site Scripting)** à l’aide de la plateforme **DVWA
(Damn Vulnerable Web Application)**.

**Types de XSS étudiés :**
- XSS Reflected
- XSS Stored

**Objectifs spécifiques :**
- Comprendre le fonctionnement des attaques XSS
- Exploiter les failles dans un environnement sécurisé
- Observer l’impact des attaques sur une application web
- Sensibilisation aux mécanismes de protection contre les XSS

**Technologies utilisées :**
- DVWA
- Apache
- Navigateur web

---

### 3️⃣ TP : Création d’un CV basique en HTML

**Description :**  
Ce TP consiste à créer un **CV simple** en utilisant uniquement **HTML**, sans
CSS, afin de maîtriser la structure et la sémantique des pages web.

**Objectifs spécifiques :**
- Utilisation correcte des balises HTML
- Structuration logique d’un document web
- Présentation claire des informations

**Technologies utilisées :**
- HTML
