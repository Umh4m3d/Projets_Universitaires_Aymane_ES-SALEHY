# Dépôt des Travaux Pratiques et Projets

Ce dépôt regroupe l’ensemble des travaux pratiques (TP) et projets réalisés dans le cadre des modules **Réseaux, Développement Web et PHP**, incluant les TPs sur Cisco Packet Tracer, DHCP, Subnetting, VLSM, et les TP PHP.  

Chaque TP est organisé dans un **dossier dédié** avec son propre code, fichiers de configuration et captures d’écran.  
Ce README présente l’organisation générale et décrit le contenu de chaque dossier.

---

## Organisation du dépôt

| Dossier | Contenu |
|--------|---------|
| `Autres_WEB` | TPs Web : création de carte d’étudiant (HTML/CSS), exploitation de failles XSS avec DVWA, création de CV basique en HTML |
| `Autres_NETWORK` | TPs Réseau : rapports sur l’adressage et la division de réseau, calcul de sous-réseaux, DHCP sous Ubuntu |
| `TP_PHP` | TP PHP : pages dynamiques, manipulation de tableaux, connexion sécurisée à MySQL via PDO, vulnérabilité LFI et protection |
| `TP_Subnetting_avec_Cisco_Packet` | TP Subnetting: calculs d’adresses IP, sous-réseaux, broadcast, gateways, architecture réseau dans Cisco Packet Tracer et tests ping |
| `Projet Réseau Universitaire` | Projet complet : architecture réseau multi-bâtiments avec segmentation VLAN, routage inter-VLAN, configuration de DHCP et tests de connectivité |
| `TP_VLSM_CISCO` | Variante TP Subnetting avec VLSM: calculs d’adresses IP, sous-réseaux, broadcast, gateways, architecture réseau dans Cisco Packet Tracer et tests ping|

---

## Consignes de dépôt

1. **Structure du dépôt**  
   - Chaque TP/projet est placé dans son propre dossier.  
   - Les fichiers source, rapports et captures d’écran sont clairement séparés.  

2. **README par dossier**  
   - Chaque dossier contient un README décrivant le TP/projet, les objectifs, les fichiers et instructions d’exécution.  

3. **Fichier principal README**  
   - Ce README centralise l’information sur tous les TP et projets et permet au professeur d’avoir une vue globale.
