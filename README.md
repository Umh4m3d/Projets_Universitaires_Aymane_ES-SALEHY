# 📁 Dépôt des Travaux Pratiques et Projets

Ce dépôt regroupe l’ensemble des travaux pratiques (TP) et projets académiques réalisés à  
l’**Université Euromed de Fès**, dans plusieurs modules, notamment :

- Réseaux  
- Développement Web sécurisé
- Cybersécurité  
- Systèmes distribués  
- Virtualisation & Cloud  

Chaque module est organisé dans un **dossier dédié**  et ce README fournit une vue globale du contenu du dépôt.

---

## 📂 Organisation du dépôt

| Dossier | Contenu |
|--------|---------|
| `Autres_WEB` | TPs Web : carte d’étudiant (HTML/CSS), CV en HTML, exploitation de failles XSS (DVWA) |
| `Autres_NETWORK` | TPs Réseau : adressage IP, subnetting, calculs de sous-réseaux, DHCP sous Ubuntu |
| `TP_PHP` | TPs PHP : pages dynamiques, tableaux, connexion MySQL avec PDO, vulnérabilité LFI et sécurisation |
| `TP_Subnetting_avec_Cisco_Packet` | Subnetting : IP, broadcast, passerelles, architecture réseau et tests |
| `TP_VLSM_CISCO` | VLSM : optimisation d’adressage IP et conception réseau |
| `Projet Réseau Universitaire` | Projet complet : réseau multi-bâtiments, VLAN, routage inter-VLAN, DHCP |
| `TPs_Distributed_Applications` | Systèmes distribués : API HTTP Python, client/serveur, JSON, versioning, sérialisation |
| `TPs_Virtualization & Cloud` | Virtualisation & Cloud : ESXi, vCenter, Veeam, Docker, Kubernetes |

---

## 📌 Organisation des TPs
- Chaque module ou projet est dans un dossier indépendant  
- Chaque dossier contient un README décrivant :
  - Objectifs  
  - Technologies utilisées  
  - Instructions d’exécution  

### 🔹 README principal
- Donne une vue globale  
- Facilite la navigation

## 🚀 Compétences développées

- **Réseaux** : Subnetting, VLSM, VLAN, DHCP
